About Us

  • Current version : v 1.1
  • Authors : Irene Komal P, Anirudh B
  • Check out the Github repo!

Dice for Python3 looks to provide developers with a fully-deployable, trainable and test-able computational offloading framework that profiles energy, runtime and CPU statistics to output comprehensive decisions using a specialised Cascade algorithm.

Requirements

The following are necessary (but not exhaustive) dependencies and requirements for Dice

  1. A Windows, MacOS or other UNIX-based operating system.
  2. Python (v3.x)
  3. A reachable server with API endpoints to configure functions for training and testing

Please note: DICE does not provide support for initial server setup, we recommend looking at AWS EC2, Microsoft Azure or a RaspberryPi-based server.
We do provide documentation on how to set up the API endpoints once you have access to a server.
